Published: 2016-08-10 (draft)

This profile defines the expected capabilities of the Data Source actor when conforming to the DAF-Research IG.

General

FHIR Version: 1.0.0
Accepts elements from future versions: extensions
Supported formats: xml, json
Supported profiles:

REST behavior

The Data Source SHALL support the DAF-Task profile, the daf-extract-operation and SHOULD support the US-Core profiles

Security:

Data Source SHALL use appropriate security protocols for data extraction.
