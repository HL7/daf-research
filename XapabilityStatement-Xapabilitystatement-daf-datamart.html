Data Mart

(Requirements Definition)

Published: 2016-08-10 (draft)

Published by: U.S. Office of the National Coordinator (ONC)

This profile defines the expected capabilities of the Data Mart actor when conforming to the DAF-Research IG.

General

FHIR Version: 1.0.0
Accepts elements from future versions: extensions
Supported formats: xml, json
Supported profiles:

REST behavior

The Data Mart SHALL support the DAF-Task, DAF-Capability, DAF-OperationDefinition profiles along with the daf-load-operation

Security:

Data Mart SHALL use appropriate security protocols for data tranlation and loading.
